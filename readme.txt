Upload To Youtube allows admin to upload and delete videos to his connected google account

== Description ==

Upload To Youtube plugin provides WordPress admin to attach his YouTube account in the dashboard seamlessly. Admin can new videos to his "Uploads" playlist as well as remove videos from it.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/upload-to-youtube` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the "UTY" menu on left sidebar to configure the plugin
